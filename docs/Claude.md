# Focus Lab – Claude Context

You are assisting development of an iOS app called "Focus Lab".

## Product Goal
Focus Lab is a game-based system designed to train executive function in children aged 5–7.

Core abilities:
- Step memory
- Sequential execution
- Focus control
- Task completion

## Tech Stack
- SwiftUI
- MVVM architecture
- No backend (local storage only)
- iPad + iPhone support

## Core Gameplay
- Show 2–5 steps
- Hide instructions
- User taps items in correct order
- Correct → advance
- Wrong → soft reset
- Success → next level

## Design Principles
- Low stimulation
- Clean UI
- Predictable interactions
- No overwhelming animations

## Constraints
- Do NOT over-engineer
- Do NOT introduce backend
- Do NOT add unnecessary features
- Focus on MVP only

## Code Style
- Keep files modular
- Avoid large monolithic views
- Prefer simple, readable SwiftUI

##Visual Design guidence 
- Use VisualSystem_Core.md as the source of truth for all UI implementation
- Ignore VisualSystem_Vision.md for implementation unless explicitly instructed.
