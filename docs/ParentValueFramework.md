# FocusLab – Parent Value Framework

## 1. Purpose

Define how FocusLab communicates its value to parents.

This framework ensures:
- value is credible (within evidence boundaries)
- messaging is clear and consistent
- product experience aligns with parent expectations

---

## 2. Core Positioning

FocusLab is NOT:
- a therapy
- a clinical treatment
- a medical solution

FocusLab IS:

A guided daily practice tool that helps children:
- build focus habits
- follow simple instructions
- complete tasks through structured repetition

---

## 3. Parent Role (Critical)

Parents are the primary decision-makers.

Their needs:
- understand what the child is practicing
- control how long and how often
- see results clearly
- feel the product is helping development

---

## 4. Value Pillars

### 4.1 Structured Practice

FocusLab is not random play.

It provides:
- step-by-step instructions
- guided interactions
- clear session boundaries

Parent takeaway:

"My child is doing structured practice, not just playing."

---

### 4.2 Short, Consistent Sessions

FocusLab emphasizes:

- short sessions (3–10 min)
- repeatable daily routines
- gradual increase over time

Parent takeaway:

"This fits into our daily routine."

---

### 4.3 Measurable Outcomes

Each session provides:

- duration practiced
- levels completed
- error count / accuracy
- completion status

Parent takeaway:

"I can see what was done and how well."

---

### 4.4 Parent Control

Parents can:

- set session duration
- enable / disable themes
- pause or end sessions

System provides recommendations, but does not enforce.

Parent takeaway:

"I am in control."

---

### 4.5 Positive Reinforcement

Children receive:

- encouraging feedback
- completion rewards
- low-pressure correction

Parent takeaway:

"My child is motivated, not stressed."

---

## 5. Safe Value Claims

FocusLab can say:

- supports focus practice
- helps children follow instructions
- encourages task completion
- builds daily practice habits

---

## 6. Claims to Avoid

Do NOT say:

- improves ADHD symptoms
- treats autism
- clinically proven effectiveness
- therapy or intervention

---

## 7. Parent-Facing Language

Use:

- "practice"
- "build"
- "support"
- "routine"
- "guided"

Avoid:

- "treatment"
- "fix"
- "cure"
- "guarantee improvement"

---

## 8. Experience Mapping

### Before Session

Parent sees:
- recommended duration
- selected themes

Message:
"This is a short guided practice session."

---

### During Session

Parent understands:
- session is active
- time is controlled
- child is following instructions

---

### After Session

Parent sees:
- summary
- completion
- basic performance

Message:
"This is what your child practiced."

---

## 9. Long-Term Value

FocusLab builds:

- consistency
- routine
- engagement with structured tasks

NOT guaranteed cognitive transformation.

---

## 10. Key Principle

FocusLab does not claim to transform ability.

It supports practice that may contribute to development over time.
