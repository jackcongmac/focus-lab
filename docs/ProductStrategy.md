# Focus Lab – Product Strategy

## 1. Product Positioning
- 3-minute daily focus training for kids
- Parent-guided, not free play
- Calm, low-stimulation, repeatable

## 2. Core Value
- Improve attention
- Build routine
- Provide structured interaction between parent and child

## 3. Usage Model
- Daily session (3–5 min)
- Parent-guided usage
- Repeatable routine

## 4. Open Questions
- How to measure improvement?
- How to motivate repeated usage?
- What is the ideal session length?
- When does child start self-initiating?

## 5. Future Systems (Do NOT build yet)
- Session system
- Progress tracking
- Theme packs
- Subscription model
