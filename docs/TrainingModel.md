# Focus Lab – Training Model

## Session Concept
- 1 session = 5–8 levels
- duration: ~3 minutes

## Skill Progression
- Level 1–4: recognition
- Level 5–8: sequence
- Level 9–10: attention filtering

## Future
- Weekly progression
- Adaptive difficulty

## Future Expansion – Item Library

- The system supports more than 4 shapes
- However, each level will only display 4 items at a time
- Different levels can swap in different item sets
- This allows content expansion without increasing cognitive load
